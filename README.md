
# 👨🏻‍💻 Customer Behavior Data Analyst Portfolio Project

This project represents a complete, industry-standard, end-to-end data analytics workflow designed to reflect real-world responsibilities of a data analyst. It covers all stages from data preparation to insight generation and visualization.

---

## 📌 Project Overview

The goal of this project is to simulate a real business analytics workflow by transforming raw data into meaningful insights.

### Key Steps:

* **Data Preparation & EDA (Python)**
  Clean and transform raw datasets for analysis

* **Data Analysis (SQL)**
  Perform queries to extract insights on customer behavior, segments, and trends

* **Visualization (Power BI)**
  Build an interactive dashboard to highlight patterns and support decision-making

* **Reporting**
  Summarize findings and provide business recommendations

---

## 🛠️ How to Use This Project

### 1. Clone the repository

```bash
git clone https://github.com/amlanmohanty1/customer-trends-data-analysis-SQL-Python-PowerBI.git
cd customer-trends-data-analysis-SQL-Python-PowerBI
```

### 2. Open Python Notebook

Open:

* `Customer_Shopping_Behavior_Analysis.ipynb`

This includes:

* Data import
* Data exploration
* Data cleaning
* SQL connection setup

---

### 3. Load Data into SQL

* Create a database (MySQL / PostgreSQL / SQL Server)
* Run Python script to load data
* Open `customer_behavior_sql_queries.sql`
* Execute queries to answer business questions

---

### 4. Connect to Power BI

* Open `customer_behavior_dashboard.pbix`
* Build and explore interactive dashboard

---

### 5. Create Report

* Prepare project report
* Create presentation for insights
